#Tarea 5

#dependencias usadas
    -dotenv
    -express 
    -mongoose
    -node
    -swagger
    -nodemon

#Comandos
    -npm install: instalar todas las dependencias
    -npm run dev: Correr el proyecto y que se reinicie solo ante cualquier cambio
    -npm start: para correr el proyecto

#Pruebas
    -Use postman para la ejecución de cada endpoint:
        -GET localhost:3000/tareas
        -GET localhost:3000/tareas/id
        -POST localhost:3000/tareas
        -PUT localhost:3000/tareas/id
        -DELETE localhost:3000/tareas/id

#Documentación del API
    -en la dirección "localhost:3000/swagger" se encuentra la documentacion de los endpoints
