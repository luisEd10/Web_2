Proyectos

Juego de la serpiente

  • Descripcion: creación de juego de la serpiente puede implicar el desarrollo de un juego en el que los usuarios controlan una serpiente y deben guiarla a través de un espacio lleno de obstáculos, mientras se alimenta de pequeños objetos que aparecen en la pantalla.
  • Tecnologías utilizadas:
    • El lenguaje de programación Java
    • La biblioteca gráfica Swing
    • Eventos de teclado y ratón
    • Algoritmos de detección de colisiones
    • Algoritmos de generación de alimentos

Pagina web de streaming para peliculas

  • Descripción: diseño y desarrollo de una plataforma en línea que permita a los usuarios ver películas en línea a través de una suscripción o compra individual. Esto podría incluir la implementación de un sistema de registro y inicio de sesión de usuarios, una interfaz de usuario atractiva y fácil de usar, la implementación de un sistema de recomendación de películas. También podría requerir el desarrollo de una base de datos robusta para almacenar información sobre las películas disponibles, así como el diseño y desarrollo de un backend escalable y seguro para manejar las solicitudes de los usuarios.
  • Tecnologías utilizadas:
    • Lenguajes HTML, CSS, JavaScript
    • Bases de datos MongoDB
    • Sistemas de control de versiones Git
    • Bibliotecas y APIs para el manejo de videos y multimedia
